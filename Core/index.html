功能说明

    Core F405：舍弃主控板所有IO接口，只保留烧录、CAN1、CAN2、USART2、USART3、UART5、I2C2 7个功能接口，20mm*40mm尺寸提高空间利用率；

    Core F042：CAN Station项目定制，保留SWD、CAN、USART、I2C 4种功能接口，额外添加4个2Pin CAN接口，20mm*40mm尺寸提高空间利用率。
